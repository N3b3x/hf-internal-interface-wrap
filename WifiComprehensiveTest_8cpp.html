Detailed Description

Interface-only WiFi testing suite for ESP32-C6 DevKit-M-1 with ESP-IDF v5.5f.

This test suite provides interface testing of the EspWifi implementation focusing on class structure, data types, and interface validation without calling ESP-IDF WiFi functions that cause linker conflicts.
