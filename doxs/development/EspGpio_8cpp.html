Detailed Description

Production-quality ESP32 GPIO implementation with ESP-IDF v5.5+ advanced features.

This file contains a world-class implementation of MCU-specific GPIO for the BaseGpio class with comprehensive ESP32/ESP-IDF v5.5+ feature support. Features include dynamic mode switching, advanced glitch filtering, RTC GPIO, power management, sleep configuration, wake-up support, interrupt handling, and comprehensive diagnostics for industrial-grade applications.
