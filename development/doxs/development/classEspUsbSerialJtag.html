◆ Default()

EspUsbSerialJtag & EspUsbSerialJtag::Default ( const hf_usb_serial_jtag_config_t & cfg = {})
staticnoexcept

Get-or-create a process-wide shared instance.

The first call configures the singleton with cfg. Subsequent calls ignore cfg and return the same instance. This matches the underlying hardware: the USB Serial/JTAG block is global.

Parameters
cfgConfiguration applied on first call only.
Returns
Reference to the shared instance.
