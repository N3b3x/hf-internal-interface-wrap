Detailed Description

template<typename Backend = PlatformMutexActiveBackend>
class PlatformSharedMutexImpl< Backend >

Reader-writer mutex using two regular mutexes + atomic counters.

Multiple readers can hold the lock concurrently; writers have exclusive access. When Backend = NullMutexBackend, all operations compile away.

Template Parameters
BackendStatic policy class satisfying the NullMutexBackend contract.
