Detailed Description

Abstract base class for built-in USB Serial/JTAG controller wrappers.

Mirrors the lazy-initialise / Read / Write / Flush shape of BaseUart so a ConsoleSerialTransport can hold either a BaseUart* or a BaseUsbSerialJtag* behind a tiny adapter.

Note
Not inherently thread-safe. The underlying ESP-IDF driver is safe for concurrent reader/writer callers, but if you wrap it with stateful helpers (e.g. printf) protect them with a mutex.
