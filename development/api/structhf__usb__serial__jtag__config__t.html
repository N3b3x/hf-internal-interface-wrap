Detailed Description

Configuration for a USB Serial/JTAG controller instance.

Intentionally minimal: the controller has no electrical knobs. Only the host-side ring-buffer sizes matter, plus a couple of policy bits.

◆ non_blocking_when_disconnected

bool hf_usb_serial_jtag_config_t::non_blocking_when_disconnected {true}

When true, calls to Read() will not block longer than timeout_ms even if the host is not connected. When false the implementation may busy-poll connection state. Default true matches typical console use.
