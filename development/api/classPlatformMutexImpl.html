Detailed Description

template<typename Backend = PlatformMutexActiveBackend>
class PlatformMutexImpl< Backend >

Recursive mutex templated on a backend policy.

When Backend = NullMutexBackend, the entire class compiles to nothing (1-byte handle member, all methods are no-ops). When a real RTOS backend is injected, this provides thread-safe recursive locking.

Template Parameters
BackendStatic policy class satisfying the NullMutexBackend contract.
